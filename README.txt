Программа была создана для сбора идей учеников насчёт улучшения школы, проведения различных мероприятий,
создания кружков.
При начале работы бот проветствует пользователя и предлагает ему написать своё предложение. Сообщение пользователя бот
пересылает администратору. Если администратор отвечает на сообщение пользователя, то ответ администратора отправиться
этому пользователю.
Администратор с помощью команды "/ban" может ограничить пересылку сообщений для определённых пользователей. Для этого
ему нужно зажать сообщение необходимого пользователя, выбрать пункт "ответить" и написать "/ban". Для разблокировки пользователя точно также
зажать сообщение, выбрать пункт "ответить" и написать "/unban". Список заблокированных пользователей сбрасывается при перезагрузке программы.
